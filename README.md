#Predef
To use [Predef](https://github.com/boostorg/predef) in [ZPM](http://zpm.zefiros.eu), just use:

# [ZPM](http://zpm.zefiros.eu) Installation
In `.package.json`
```json
"requires": [
    {
        "name": "Zefiros-Software/Predef",
        "version": "^1.66.0"
    }
]
```

In `zpm.lua`
```lua
zpm.uses "Zefiros-Software/Predef"
```
